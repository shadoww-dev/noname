✨ Introduction


Just a brief summary in question of the bot's creation The initiative to create this bot arose from a necessity. As soon as I discovered the programming language, I started studying with a lot of focus and thus being able to get to where I am now. Currently there are few functions, post I add only when necessary. Something like to streamline my work in everyday life. Everyone around me talks about investing in it, but so far I only see it as a fun, develop solution and etc. I believe that my bot is not 100% complete or bug-free, when I have time I stop to test everything and see if it continues to work right.
